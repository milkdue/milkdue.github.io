butterfly分类页面魔改

发表于2021-09-17|更新于2023-03-08|学习&教程
|字数总计:1.4k|阅读时长:8分钟|阅读量:

觉得分类页面不是很好看，打算魔改一下分页页面，恰好店长有写过首页分类的磁贴插件，于是打算利用店长的插件中的配置来进行魔改一番(具体效果可查看分类页面)，下面记录一下魔改步骤~

修改源码

  1. 修改 [Blogroot]\themes\butterfly\layout\includes\page\categories.pug

  2. 新建 [Blogroot]\themes\butterfly\scripts\helpers\category.js

  3. 新建 [Blogroot\themes\butterfly\source\css\costom.css]

  4. 在 _config.butterfly.yml.yml 的 inject 中引用 css

  5. 如果使用店长的插件，不需要做任何配置即可在分类页面中看到不一样的分类，如果没有使用店长的插件，可以在 _config.butterfly.yml.yml 配置以下内容
